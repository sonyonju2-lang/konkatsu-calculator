# 未婚人口統計

公的統計を基に、年齢・性別・地域などの条件から  
未婚人口の該当人数および割合を推定・可視化するWebツールです。

## 概要
- 対象：18歳以上の未婚者（離別・死別・配偶関係不詳を除く）
- 主目的：条件別に「市場規模」を把握するための参考情報の提示
- 本ツールは診断・判定・評価を目的とするものではありません

## データ出典
- 未婚人口：令和2年 国勢調査
- 年収分布（推定）  
  - 国税庁「民間給与実態統計調査」  

※ 年収は性別・年齢階層別の統計分布を基にした推定値であり、  
　個人の実際の収入を示すものではありません。

## 推定について
- 本ツールで表示される数値は、公開されている公的統計を基にした**推定値**です
- 実測値・個別の状況・将来の結果を保証するものではありません
- 統計の制約上、簡略化・仮定を含みます
